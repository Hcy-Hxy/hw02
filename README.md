## 任务二：Chatbot 示例代码

### 采用的 API/平台
- API 类型：DeepSeek API
- 平台：DeepSeek 官方 API

### 功能说明
- 基于DeepSeek API实现的简单聊天机器人
- 支持实时对话交互
- 包含错误处理机制
- 支持通过环境变量或直接修改代码配置API Key

### 运行方式
1. 安装依赖：pip install -r requirements.txt
2. 配置 API Key：
   - 方法 1：设置环境变量 DEEPSEEK_API_KEY
   - 方法 2：直接在代码中修改 API_KEY 变量
3. 运行命令：python chatbot_deepseek.py
4. 交互方式：输入消息后按回车，输入exit退出
